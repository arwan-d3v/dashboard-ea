# Dashboard EA - Expert Advisor Management System

Modern dashboard untuk mengelola Expert Advisor (MT5/MQL5), License, User, Approval, dan Analytics real-time.

## ✨ Fitur Utama

- **EA Manager** dengan real-time control & terminal log
- **License Manager** + Maker-Checker Approval
- **User Management** dengan RBAC
- **Analytics** dengan heatmap MQL5 snapshot
- **Live WebSocket** VPS monitoring
- Firebase Auth + Firestore
- Dark mode + Responsive

## 🛠 Tech Stack

- **Next.js 15** (App Router)
- Tailwind CSS v4
- Firebase (Auth + Firestore)
- React + JavaScript

## 🚀 Quick Start

```bash
git clone https://github.com/arwan-d3v/dashboard-ea.git
cd dashboard-ea
npm install
cp .env.example .env.local
npm run dev
```
